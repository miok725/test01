Font

  • - Color : #111(default)
  • - point Color : #3d9bff , #f4670a, #3b5997
  • - size (default) : 14px
  • - font (Noto Sans : 2가지 사용) : font-family:"NotoKR","맑은 고딕","돋움",dotum,sans-serif;
    (기본) NotoSansKR-Regular : font-family:"NotoKR";
    (bold) NotoSansKR-Medium : font-weight:bold;

image -네이밍

  • - 자유롭게 네이밍 하되 특정 prefix등을 사용하여 그룹별로 묶일수 있는 네이밍 권장
  • - title: _tit_
  • - background: bg_
  • - icon: ico_
  • - bullet: dot_ , arr_ , bar_
  • - button: btn_
  • - tab: _tab_

class -네이밍 (camel 기법사용)

  • - 자유롭게 네이밍 하되 특정 prefix등을 사용하여 그룹별로 묶일수 있는 네이밍 권장
  • - wrap: wrapTblDefault / stepListWrap / popWrap
  • - title: defTit / formTit / subTit
  • - txt: defTxt / infoTxt / infoTxtCaution
  • - list: listDot / listDash / stepListWrap
  • - icon: bankIconList / scOtpIcon
  • - bullet: inputDot /
  • - button: btnPressDefault / btnBodyDefault / btnFuncTip
  • - tab: tabLv0 / tabInner / tabSwitchCont
