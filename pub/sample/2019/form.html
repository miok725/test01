form타입 : input/select 혼합형

input 혼합형의 활용

  • div.inputBoth01 에 원하는 비율에 따라 클래스 추가
    * 비율클래스 : div2_8, div3_7, div4_6, div5_5, div6_4, div7_3, div8_2
  • 좌측에 보여야 하는 요소에 inputB01 클래스를 넣고
  • 우측에 보여야 하는 요소에 inputB02 클래스를 넣는다.

inputBoth01, inputBoth02, inputBoth03 의 사용

  • inputBoth01 : 정렬되어야 하는 요소가 두개일 경우 사용(가장 많이 사용됨)
  • inputBoth02 : 정렬되어야 하는 요소가 50% 로 고정되어야 할 경우 사용
  • inputBoth03 : 정렬되어야 하는 요소 사이에 dash( - 혹은 / ) 나 dot( . ) 가 함께 사용되는 경우
